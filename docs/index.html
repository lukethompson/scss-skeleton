    mixins

    position

    @mixin position($position, $top: null, $right: null, $bottom: null, $left: null) { ... }

    Description

    Shorthand mixin for offset positioning

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $position

    Either relative, absolute or fixed

    		String— none
    $top

    Top offset

    		Lengthnull
    $right

    Right offset

    		Lengthnull
    $bottom

    Bottom offset

    		Lengthnull
    $left

    Left offset

    		Lengthnull

    Example

    Position element .foo with the position mixin

    .foo {
  @include position(fixed, 0, 0, 0, 0)
}

    respond-to

    @mixin respond-to($size) { ... }

    Description

    Helper mixin for responding to screen width changes

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $size

    Either sm, md, lg or xl

    		String— none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    Apply respond to mixin to .foo

    .foo {
  color: #fff;
  @include respond-to(md) {
    color: #000;
  }
}

    Requires

    • [variable] sm-width
    • [variable] md-width
    • [variable] lg-width
    • [variable] xl-width

    None.
